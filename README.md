# Similaridade de Sentenças com NLP

Projeto de estudo em **Processamento de Linguagem Natural (NLP)** com foco no cálculo de **similaridade entre sentenças**, utilizando técnicas de vetorização de texto e modelos de linguagem distribuída.

O notebook explora desde o pré-processamento textual até a representação semântica das frases, aplicando conceitos fundamentais de NLP para comparação de textos.

## Objetivo

Analisar a similaridade semântica entre sentenças de avaliações textuais, investigando como modelos de linguagem podem capturar relações de significado entre frases diferentes.

## Base de Dados

- Dataset de avaliações textuais (`reviews_zoop.csv`).
- Textos utilizados para experimentação e comparação de similaridade entre sentenças.
- Base obtida a partir de material educacional.

## Etapas do Projeto

- Carregamento e exploração inicial dos dados.
- Tokenização de textos com **NLTK**.
- Pré-processamento linguístico.
- Treinamento de modelo **Word2Vec** e **Doc2Vec**com **Gensim**.
- Representação vetorial de palavras e sentenças.
- Cálculo de similaridade semântica entre frases.
- Análise dos resultados obtidos.

## Tecnologias e Bibliotecas Utilizadas

- **Python**
- **Pandas** – manipulação de dados
- **NLTK** – tokenização e recursos linguísticos
- **Gensim** – Word2Vec e Doc2Vec
- **Jupyter Notebook**

## Conceitos Explorados

- Processamento de Linguagem Natural (NLP).
- Similaridade semântica.
- Word Embeddings.
- Word2Vec.
- Doc2Vec.
- Análise de textos.
- Representação vetorial de linguagem natural.

Study project in **Natural Language Processing (NLP)** focused on computing **sentence similarity**, using text vectorization techniques and distributed language models.

The notebook explores the full pipeline, from textual preprocessing to semantic sentence representation, applying fundamental NLP concepts for text comparison.

## Objective

To analyze the semantic similarity between sentences from textual reviews, investigating how language models can capture meaningful relationships between different sentences.

## Dataset

- Textual reviews dataset (`reviews_zoop.csv`).
- Texts used for experimentation and sentence similarity comparison.

## Project Steps

- Data loading and initial exploration.
- Text tokenization using **NLTK**.
- Linguistic preprocessing.
- Training **Word2Vec** and **Doc2Vec** models with **Gensim**.
